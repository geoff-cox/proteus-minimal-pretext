    Exercise 36. Initial Value Problem Solution.

    Which of the following is a solution to the initial value problem
    \begin{equation*} \frac{dy}{dx} = 2y - 10,\quad y(0) = 8\text{?} \end{equation*}
    üîó
    • \(y(x) = 3e^{2x} + 5\)
    • Correct! This function satisfies the differential equation since
      \begin{align*} \frac{d}{dx}\left[3e^{2x} + 5\right] \amp = 2[3e^{2x} + 5]-10 \\ 6e^{2x} \amp = 6e^{2x} + 10 - 10 \\ 6e^{2x} \amp = 6e^{2x} \quad ‚úÖ \end{align*}
      and the initial condition, since \(y(0) = 3(1) + 5 = 8 \quad ‚úÖ\text{.}\)
    • \(y(x) = e^{2x} + 5\)
    • This does not satisfy the initial condition since \(y(0) = 1 + 5 = 6 \neq 8 \quad ‚ùå\text{.}\)
    • \(y(x) = 2e^{2x} + 6\)
    • This satisfies the initial condition since \(y(0) = 2(1) + 6 = 8 \quad ‚úÖ\text{,}\) but it does not satisfy the differential equation since
      \begin{align*} \frac{d}{dx}\left[2e^{2x} + 6\right] \amp = 2[2e^{x} + 6]-10 \\ 4e^{2x} \amp = 4e^{2x} + 12 - 10 \\ 4e^{2x} \amp = 4e^{2x} + 2 \quad ‚ùå \text{.} \end{align*}
