    Exercise 40. Range of a sequence.

    Select all statements that correctly describe the \(\text{range}\) of a sequence \(\{a_n\}\text{.}\)
    üîó
    • It is the set of all distinct values taken on by \(a_n\text{.}\)
    • The range collects values the sequence hits, ignoring how many times a value repeats.
    • It is the ordered list \(a_1,a_2,a_3,\dots\text{.}\)
    • The ordered list is the list of terms. The range is a set, so it ignores order and repetition.
    • If \(a_n\) repeats values, those repeated values appear only once in the range.
    • Repetition does not create new elements of the range.
    • The range must be infinite because the sequence has infinitely many terms.
    • A sequence can take only finitely many distinct values (for example, alternating between two numbers).
    • The range can be finite even though the domain \(\mathbb{N}\) is infinite.
    • What matters is how many distinct outputs occur, not how many inputs there are.
    • Changing the order of the terms changes the range.
    • The range is a set of values, so order does not matter; reordering terms does not change which values occur.
    üîó

    Exercise 41. Using the function‚Äìsequence limit link.

    Suppose you find a function \(f(x)\) with \(f(n)=a_n\) for all \(n\in\mathbb{N}\text{,}\) but the limit \(\displaystyle\lim_{x\to\infty} f(x)\) does not exist. What can you conclude about \(\displaystyle\lim_{n\to\infty} a_n\text{?}\)
    üîó
    • The sequence limit does not exist.
    • A non-existent limit for \(f(x)\) over the reals does not force the integer-input subsequence \(f(n)\) to behave the same way.
    • No definite conclusion: the sequence limit may exist or may not exist.
    • It is possible for \(f(x)\) to oscillate on real inputs while \(f(n)\) behaves more simply on integer inputs (for example, a cosine function can do this).
    • The sequence must diverge to \(\infty\text{.}\)
    • Diverging to \(\infty\) is a specific type of divergence; ‚Äúno limit for \(f(x)\)‚Äù does not imply unbounded growth of \(a_n\text{.}\)
    • The sequence must converge, but you cannot find the value.
    • The theorem only gives implications in one direction: a limit for \(f(x)\) implies a limit for \(a_n\text{,}\) not the other way around.

    Exercise 43. Convergent vs. bounded.

    Which statement is always true?
    üîó
    • Every bounded sequence converges.
    • A sequence can stay within bounds and still fail to settle to one value (oscillation is a common reason).
    • Every convergent sequence is bounded.
    • If the terms get arbitrarily close to a finite limit, then all but finitely many terms lie in a small interval around that limit, and the remaining finitely many terms can also be bounded.
    • Every divergent sequence is unbounded.
    • Divergence can happen even when the terms remain between two numbers (for instance, by oscillating without approaching a single value).
    • If a sequence is unbounded, then it must diverge to \(\infty\text{.}\)
    • ‚ÄúUnbounded‚Äù allows growth toward \(\infty\text{,}\) toward \(-\,\infty\text{,}\) or even behavior that escapes in both directions.

    Exercise 47. Understanding Limits of Sequences.

    Which of the following best describes what it means for a sequence \(\{a_n\}\) to have limit \(L\text{?}\)
    üîó
    • Every term of the sequence equals \(L\text{.}\)
    • This is too restrictive. A sequence can converge to \(L\) even if no term actually equals \(L\text{,}\) as long as the terms get arbitrarily close to \(L\text{.}\)
    • The terms of the sequence get closer to \(L\text{,}\) but never reach it.
    • This is close, but not quite right. The terms can actually equal \(L\) sometimes‚Äîthe key is that they eventually stay arbitrarily close to \(L\text{.}\)
    • For any \(b \ne L\text{,}\) all terms past some point will be closer to \(L\) than \(b\text{.}\)
    • Excellent! This captures the essence of the \(\varepsilon\)-\(N\) definition:
      Given any \(\varepsilon > 0\) \(\leftarrow\) ‚Äúdistance between \(L\) and \(b\)‚Äù

      such that \(|a_n - L| < \varepsilon\) \(\leftarrow\) ‚Äú\(a_{N}, a_{N+1}, a_{N+2}, \ldots\) are closer to \(L\) than \(b\)‚Äù
      for all \(n > N\) \(\leftarrow\) ‚Äúfor all terms past some point‚Äù
      \(\)
    • The average of all terms in the sequence equals \(L\text{.}\)
    • This describes a different concept (Ces√†ro summation). Limits are about the eventual behavior of terms, not their average.

    Exercise 48. Connecting Function and Sequence Limits.

    • \(\lim\limits_{n\to\infty} a_n\) might equal 3, but we need more information.
    • We actually have enough information. Check Theorem 8.1.2 about the relationship between function limits and sequence limits.
    • \(\lim\limits_{n\to\infty} a_n = 3\)
    • Correct! By Theorem 8.1.2, if \(\lim\limits_{x\to\infty} f(x) = L\) and \(f(n) = a_n\text{,}\) then \(\lim\limits_{n\to\infty} a_n = L\text{.}\) This theorem is extremely useful for evaluating sequence limits!
    • The sequence diverges.
    • No‚Äîsince the corresponding function limit exists and equals 3, the sequence converges to 3 by Theorem 8.1.2.

    Exercise 49. Convergence and Divergence Properties.

    Which of the following statements are TRUE?
    üîó
    • If \(\lim\limits_{x\to\infty} f(x)\) does not exist, the sequence \(\{f(n)\}\) might still converge.
    • Correct! The example \(f(x) = \cos(2\pi x)\) oscillates as a function, but \(\{f(n)\} = \{\cos(2\pi n)\} = \{1, 1, 1, \ldots\}\) converges to 1.
    • A sequence that oscillates between two values must diverge.
    • This is false. While the sequence \(\{4 + (-1)^n\}\) from Example 8.1.4 does oscillate and diverge, a sequence could oscillate between values that get closer together and still converge (like \(\{(-1)^n/n\}\) converging to 0).
    • If \(\lim\limits_{n\to\infty} |a_n| = 0\text{,}\) then \(\lim\limits_{n\to\infty} a_n = 0\text{.}\)
    • Correct! This is the Absolute Value Theorem (Theorem 8.1.4). If the absolute values approach 0, then the terms themselves must approach 0.
    • Every convergent sequence must be monotonic.
    • This is false. The sequence \(\{(-1)^n/n\}\) converges to 0 but is not monotonic‚Äîit alternates between positive and negative values.
    • We can use L‚ÄôH√¥pital‚Äôs Rule to evaluate some sequence limits.
    • Correct! When we can write \(a_n = f(n)\) for some function \(f\text{,}\) we can use L‚ÄôH√¥pital‚Äôs Rule on \(\lim\limits_{x\to\infty} f(x)\) and then apply Theorem 8.1.2.

    Exercise 53. Applying the Monotonic Convergence Theorem.

    Consider the sequence \(\{a_n\}\) where \(a_1 = 2\) and \(a_{n+1} = \sqrt{2 + a_n}\) for all \(n \geq 1\text{.}\) Suppose we can show that:
    üîó
    (i) \(a_n < 3\) for all \(n\text{,}\) and
    üîó
    (ii) \(a_{n+1} > a_n\) for all \(n\text{.}\)
    üîó
    What can we conclude?
    üîó
    • The sequence might converge or might diverge‚Äîwe need more information.
    • We actually have enough information! We know the sequence is both monotonic (increasing) and bounded (above by 3, below by 2).
    • The sequence must converge to some limit \(L\) where \(2 \leq L \leq 3\text{.}\)
    • Excellent! By Theorem 8.1.10, a monotonically increasing sequence that is bounded above must converge. Since all terms satisfy \(2 \leq a_n < 3\text{,}\) the limit must satisfy \(2 \leq L \leq 3\text{.}\) (In fact, by taking limits in the recursion relation, we can show \(L = 2\text{!}\))
    • The sequence must converge to 3.
    • While the theorem guarantees convergence, it doesn‚Äôt tell us the exact value of the limit. We know the limit is at most 3, but it could be less.
    • The sequence must diverge to infinity.
    • No‚Äîthe sequence is bounded above by 3, so it cannot diverge to infinity. Check Theorem 8.1.10 about bounded monotonic sequences.
